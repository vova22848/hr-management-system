# 🏢 АРМ Начальника отдела кадров

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Flask](https://img.shields.io/badge/Flask-2.3.3-green.svg)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5.3-purple.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

Веб-приложение для автоматизации работы начальника отдела кадров предприятия. Позволяет управлять сотрудниками, подразделениями, должностями, приказами и отпусками с удобным веб-интерфейсом.

## ✨ Основные возможности

- 📊 **Дашборд** с ключевыми показателями и графиками
- 👥 **Управление сотрудниками** (прием, увольнение, перевод)
- 🏢 **Управление подразделениями** и должностями
- 📄 **Кадровые приказы** с автоматической нумерацией
- 🏖️ **Учет отпусков** (ежегодные, учебные, декретные)
- ✈️ **Командировки** и больничные листы
- 📈 **Отчеты и аналитика** с визуализацией данных
- 💾 **Экспорт данных** в CSV для PostgreSQL
- 📱 **Адаптивный дизайн** для мобильных устройств

## 🚀 Технологии

- **Backend**: Python 3.8+, Flask 2.3.3
- **База данных**: SQLite (с возможностью миграции на PostgreSQL)
- **Frontend**: HTML5, CSS3, Bootstrap 5, Chart.js
- **Дополнительно**: Font Awesome, JavaScript (ES6+)
